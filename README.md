## FRAU — конфигуратор кухни

Адаптивный Vue 3 SPA с пошаговым конфигуратором кухни: формат (прямая/угловая), размеры помещения, фрезеровка, столешница, палитра из 32 оттенков, типы ручек и подсветка. Данные готовы к отправке в ИИ/бэкенд, когда появится API.

### Быстрый старт
```bash
npm install
npm run dev
```
Открыть: http://localhost:5173

### Сборка
```bash
npm run build
```

### Контент
- Файл обложки: `public/public-media-mebel.jpg` — положите туда фото кухни/рендера (сейчас используется заглушка-градиент).
- Основные стили: `src/style.css`
- Шаги и логика конфигуратора: `src/App.vue`
